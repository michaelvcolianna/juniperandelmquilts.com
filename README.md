# Juniper & Elm Quilts

Lani's quilting website w/homepage, blog, and contact form. Current site is on Squarespace. This is a test install to see if Storyblok/Nuxt can take the place of Squarespace.

Eventual plans:
- Source products from Etsy
- Link w/shopping cart (possibly Spotify)

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
